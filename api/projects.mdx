## Overview

Projects represent specific software products or services delivered to clients. Each project is associated with a client and has at least one license. The project endpoint allows you to create both the project and its initial license in a single transaction.

## Create Project with License

Create a new project and generate its first license automatically.

### Endpoint

```
POST /projects/createProject
```

<Warning>
  This endpoint requires authentication. Include a valid JWT token in the Authorization header.
</Warning>

### Authentication

Include the JWT token in the Authorization header:

```
Authorization: Bearer <your-jwt-token>
```

### Request Body

<ParamField body="clientId" type="string" required>
  MongoDB ObjectId of the client this project belongs to
</ParamField>

<ParamField body="projectName" type="string" required>
  Name of the project
</ParamField>

<ParamField body="duration" type="number" required>
  License duration in months (1-12)
</ParamField>

<ParamField body="services" type="array" required>
  Array of services included in the license (e.g., ["Hosting", "Domain"]). Must include at least one service.
</ParamField>

## Transaction Handling

<Note>
  This endpoint uses MongoDB transactions to ensure atomic creation of both project and license.
</Note>

The transaction ensures:

1. **Atomicity**: Both project and license are created together, or neither is created
2. **Consistency**: If any step fails, all changes are rolled back
3. **Data Integrity**: No orphaned projects or licenses

### Transaction Flow

<Steps>
  <Step title="Start Transaction">
    Begin MongoDB transaction (skipped in test environment)
  </Step>
  
  <Step title="Validate Client">
    Verify the client exists
  </Step>
  
  <Step title="Create Project">
    Insert project record within transaction
  </Step>
  
  <Step title="Generate License">
    Create license with auto-generated key within transaction
  </Step>
  
  <Step title="Commit">
    Commit transaction if all steps succeed
  </Step>
  
  <Step title="Rollback on Error">
    Abort transaction if any step fails
  </Step>
</Steps>

## Services

The `services` array defines what's included in the license. Available options:

- **Hosting**: Web hosting service
- **Domain**: Domain registration/management

You must provide at least one service:

```json
{
  "services": ["Hosting"]
}
```

Or multiple services:

```json
{
  "services": ["Hosting", "Domain"]
}
```

## Complete Example

<CodeGroup>
```javascript JavaScript
class ProjectManager {
  constructor(serverUrl, authToken) {
    this.serverUrl = serverUrl;
    this.authToken = authToken;
  }

  async createProject(clientId, projectName, duration, services) {
    try {
      const response = await fetch(
        `${this.serverUrl}/projects/createProject`,
        {
          method: 'POST',
          headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${this.authToken}`
          },
          body: JSON.stringify({
            clientId,
            projectName,
            duration,
            services
          })
        }
      );

      if (!response.ok) {
        const error = await response.json();
        throw new Error(error.message || `HTTP ${response.status}`);
      }

      const data = await response.json();
      
      console.log('Project created:', data.project.name);
      console.log('License key:', data.license.key);
      console.log('License expires:', new Date(data.license.expiresAt));
      
      return data;
    } catch (error) {
      console.error('Failed to create project:', error.message);
      throw error;
    }
  }
}

// Usage - Complete workflow
const manager = new ProjectManager(
  'https://your-keybox-server.com',
  'your-jwt-token'
);

try {
  // Assuming you already have a client
  const result = await manager.createProject(
    '60f7b3b3e6b3a72e8c8e4a1a', // clientId
    'E-commerce Website',        // projectName
    12,                          // duration (months)
    ['Hosting', 'Domain']        // services
  );
  
  // Send license key to client
```

```python Python
import requests
from datetime import datetime

class ProjectManager:
    def __init__(self, server_url, auth_token):
        self.server_url = server_url
        self.auth_token = auth_token

    def create_project(self, client_id, project_name, duration, services):
        try:
            headers = {
                'Authorization': f'Bearer {self.auth_token}'
            }
            
            response = requests.post(
                f'{self.server_url}/projects/createProject',
                headers=headers,
                json={
                    'clientId': client_id,
                    'projectName': project_name,
                    'duration': duration,
                    'services': services
                }
            )
            
            if not response.ok:
                error = response.json()
                raise Exception(error.get('message', f'HTTP {response.status_code}'))
            
            data = response.json()
            
            print(f"Project created: {data['project']['name']}")
            print(f"License key: {data['license']['key']}")
            
            expires_at = datetime.fromisoformat(
                data['license']['expiresAt'].replace('Z', '+00:00')
            )
            print(f"License expires: {expires_at}")
            
            return data
        except Exception as error:
            print(f'Failed to create project: {str(error)}')
            raise

# Usage - Complete workflow
manager = ProjectManager(
    'https://your-keybox-server.com',
    'your-jwt-token'
)

try:
    # Assuming you already have a client
    result = manager.create_project(
        '60f7b3b3e6b3a72e8c8e4a1a',  # client_id
        'E-commerce Website',         # project_name
        12,                           # duration (months)
        ['Hosting', 'Domain']         # services
    )
    
    # Send license key to client
    license_key = result['license']['key']
    print('\nShare this license key with your client:')
    print(license_key)
    
except Exception as error:
    print(f'Setup failed: {str(error)}')
```

## Workflow Integration
