## Overview

The dashboard endpoint returns a hierarchical view of all your data - clients with their projects, and each project with its licenses. This provides a complete overview of your licensing infrastructure.

## Endpoint

```
GET /dashboard
```

<Warning>
  This endpoint requires authentication. Include a valid JWT token in the Authorization header.
</Warning>

## Authentication

Include the JWT token in the Authorization header:

```
Authorization: Bearer <your-jwt-token>
```

## Request

### Optional Request Body

<ParamField body="clientId" type="string">
  Optional MongoDB ObjectId to filter dashboard data for a specific client
</ParamField>

<Note>
  If no clientId is provided, the endpoint returns data for all clients owned by the authenticated user.
</Note>

### Example Request

<CodeGroup>
```bash cURL - All Clients
curl -X GET https://your-keybox-server.com/dashboard \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

```bash cURL - Specific Client
curl -X GET https://your-keybox-server.com/dashboard \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -d '{
    "clientId": "60f7b3b3e6b3a72e8c8e4a1a"
  }'
```

```javascript JavaScript
// Fetch all clients
```

    Client's email address

  <ResponseField name="owner" type="string">
    User ID who owns this client
  </ResponseField>

  <ResponseField name="projects" type="array">
    Array of project objects
  </ResponseField>

  <Expandable title="projects structure">
    Each project object contains:

    <ResponseField name="_id" type="string">
      Project's MongoDB ObjectId
    </ResponseField>

    <ResponseField name="name" type="string">
      Project name
    </ResponseField>

    <ResponseField name="client" type="string">
      Client ID this project belongs to
    </ResponseField>

    <ResponseField name="licenses" type="array">
      Array of license objects for this project
    </ResponseField>

    <Expandable title="licenses structure">
      Each license object contains:

      <ResponseField name="_id" type="string">
        License MongoDB ObjectId
      </ResponseField>

      <ResponseField name="key" type="string">
        License key
      </ResponseField>

      <ResponseField name="duration" type="number">
        Duration in months
      </ResponseField>

      <ResponseField name="issuedAt" type="date">
        When license was created
      </ResponseField>

      <ResponseField name="expiresAt" type="date">
        When license expires
      </ResponseField>

      <ResponseField name="status" type="string">
        License status (PENDING, ACTIVE, EXPIRED, REVOKED)
      </ResponseField>

      <ResponseField name="services" type="array">
        Array of included services
      </ResponseField>

      <ResponseField name="machineId" type="string">
        Hashed machine ID (if activated)
      </ResponseField>
    </Expandable>

## Data Hierarchy

The dashboard returns data in this hierarchy:

```
Client
├─ Client Info (name, email, owner)
└─ Projects []
   ├─ Project Info (name, client)
   └─ Licenses []
      └─ License Info (key, status, duration, etc.)
```

## Query Behavior

<AccordionGroup>
  <Accordion title="User Isolation">
    The endpoint automatically filters by the authenticated user's ID (`owner` field). You only see clients you own.
  </Accordion>

  <Accordion title="Client Filtering">
    If you provide a `clientId`, only that specific client's data is returned (if you own it).
  </Accordion>

  <Accordion title="Nested Population">
    The endpoint uses MongoDB's aggregation to efficiently fetch nested relationships without multiple queries.
  </Accordion>
</AccordionGroup>

```python Python
import requests
from datetime import datetime, timedelta

class Dashboard:
    def __init__(self, server_url, auth_token):
        self.server_url = server_url
        self.auth_token = auth_token
        self.last_data = None

    def fetch_all(self):
        try:
            headers = {
                'Authorization': f'Bearer {self.auth_token}'
            }
            
            response = requests.get(
                f'{self.server_url}/dashboard',
                headers=headers
            )
            
            response.raise_for_status()
            data = response.json()
            
            self.last_data = data
            self.display_summary(data)
            
            return data
        except Exception as error:
            print(f'Failed to fetch dashboard: {error}')
            raise

    def fetch_client(self, client_id):
        try:
            headers = {
                'Authorization': f'Bearer {self.auth_token}'
            }
            
            response = requests.get(
                f'{self.server_url}/dashboard',
                headers=headers,
                json={'clientId': client_id}
            )
            
            response.raise_for_status()
            return response.json()
        except Exception as error:
            print(f'Failed to fetch client: {error}')
            raise

    def display_summary(self, data):
        print('\n=== Dashboard Summary ===')
        print(f"Total Clients: {data['clientsCount']}")
        print(f"Total Projects: {data['projectsCount']}")
        
        total_licenses = 0
        active_licenses = 0
        pending_licenses = 0
        
        for client in data['data']:
            for project in client['projects']:
                total_licenses += len(project['licenses'])
                for license_data in project['licenses']:
                    if license_data['status'] == 'ACTIVE':
                        active_licenses += 1
                    if license_data['status'] == 'PENDING':
                        pending_licenses += 1
        
        print(f'Total Licenses: {total_licenses}')
        print(f'Active: {active_licenses}')
        print(f'Pending: {pending_licenses}')
        print('========================\n')

    def get_expiring_licenses(self, days=30):
        threshold = datetime.now() + timedelta(days=days)
        expiring = []
        
        if not self.last_data:
            return expiring
        
        for client in self.last_data['data']:
            for project in client['projects']:
                for license_data in project['licenses']:
                    expires_at = datetime.fromisoformat(
                        license_data['expiresAt'].replace('Z', '+00:00')
                    )
                    if license_data['status'] == 'ACTIVE' and expires_at < threshold:
                        expiring.append({
                            'client': client['name'],
                            'project': project['name'],
                            'key': license_data['key'],
                            'expiresAt': expires_at
                        })
        
        return expiring

# Usage
dashboard = Dashboard(
    'https://your-keybox-server.com',
    'your-jwt-token'
)

# Fetch all data
data = dashboard.fetch_all()

# Check for expiring licenses
expiring = dashboard.get_expiring_licenses(30)
if expiring:
    print('\nLicenses expiring in 30 days:')
    for item in expiring:
        print(f"{item['client']} - {item['project']}: {item['expiresAt'].strftime('%Y-%m-%d')}")
```
